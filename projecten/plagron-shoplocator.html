De Shoplocator is gebouwd binnen de Laravel website van Plagron. Door middel jQuery worden de winkels uit onze zelfgebouwde microservice gehaald. Eens in de zoveel tijd doet deze microservice een verzoek aan de API van Teamleader CRM. Dit verzoek haalt alle winkels voor een bepaald land op en schrijft deze weg in JSON bestanden, zodat we later de API niet meer hoeven aan te roepen. Vervolgens wordt hier door middel van de Markerclusterer API van Google voor elke winkel een marker geplaatst op het kaartje.
